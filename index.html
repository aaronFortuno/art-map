Guia ràpida

Explora el mapa

  • Passar el ratolí sobre un node ressalta el seu veïnat i atenua la resta.
  • Clicar un node el fixa: obre la fitxa al panell i empeny els veïns cap enfora (blooming) per llegir-los millor.
  • Clicar el fons del mapa desfixa.
  • Roda del ratolí: zoom suau amb el cursor com a pivot. Les vores i les etiquetes queden a la mateixa mida; només creix la imatge.
  • Arrossegar un node el mou. La posició es desa automàticament.

Disposició

  • Xarxa (força): els nodes es col·loquen per repulsió. Les posicions es desen entre recàrregues.
  • Cronologia: X per any, Y per carril. En seleccionar un node, els veïns del seu segle es comprimeixen verticalment per llegir-se sense encavalcar-se.
  • El botó Reordena genera una nova disposició en mode xarxa.

Filtres i cerca

  • Cerca: filtra en calent per títol, autor, any, període o tema mentre escrius.
  • Tipus de connexió: 5 categories (precedent, cita, ruptura, parentiu, eco). Marca/desmarca per amagar arestes.
  • Períodes i temes: amaguen nodes fora de la selecció. Caixes plegables al panell esquerre.

Fitxa d'un node

  • Ampliar sobre la imatge obre el mode pantalla completa amb imatge gran + anàlisi lateral.
  • 📄 Exportar fitxa (PDF): genera un sol full amb aquesta obra (portada, metadades, anàlisi, connexions).
  • 🔗 Copiar enllaç: posa al porta-retalls un enllaç directe a aquesta fitxa (format #node/id).

Mode presentació

  1. Activa Mode presentació a la capçalera.
  2. Clica obres al mapa i prem + Afegir a la presentació per construir la seqüència.
  3. Reordena arrossegant diapositives; elimina amb ×.
  4. Importa / Exporta la seqüència com a JSON per reutilitzar-la.
  5. ▶ Començar obre una vista fosca a pantalla completa amb navegació.
